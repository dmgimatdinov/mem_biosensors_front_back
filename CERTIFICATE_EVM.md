# Описание для свидетельства о государственной регистрации программы для ЭВМ

---

## Название программы для ЭВМ

**Система управления паспортами мемристивных биосенсоров**

---

## Реферат

Программа предназначена для систематизации, хранения и анализа данных о мемристивных биосенсорах в научно-исследовательских и инженерных целях.

**Область применения:** разработка и исследование биосенсоров на основе мемристивных структур; ведение базы знаний по компонентам биосенсоров и их характеристикам.

**Функциональные возможности:**

- Ведение справочников компонентов биосенсора: аналитов, биораспознающих слоёв, иммобилизационных слоёв и мемристивных слоёв с заданием их физико-химических параметров и пределов определения.
- Формирование паспортов биосенсоров — структурированных записей, объединяющих все составные слои и характеристики конкретного сенсорного устройства.
- Автоматизированный синтез новых комбинаций биосенсорных структур на основе хранимых компонентов.
- Аналитические функции: расчёт показателей базы данных, построение сводных таблиц и визуализация характеристик биосенсоров.
- Экспорт данных в форматах Excel (.xlsx), CSV, PDF и DOCX.
- Парсинг патентной документации с сервиса Google Patents: автоматическая загрузка PDF-файлов патентов и извлечение ключевых данных.
- Частотный анализ данных биосенсоров с применением библиотек NumPy и Pandas.
- Веб-интерфейс пользователя, реализованный на базе фреймворка Next.js (React); серверная часть реализована с использованием фреймворка FastAPI.
- Хранение данных в реляционной базе данных SQLite.

**Тип ЭВМ:** IBM PC-совместимые ПК; x86-64. **ОС:** Linux, Windows.
